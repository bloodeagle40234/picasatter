Picasatter
+++++++++++++++++++++++

README & USAGE
=======================

This is slideshow widget of
google picasa.

It based on monitter.(twitter widget)
And we extends it to google feeds.(picasa API)

If you use twitter widget, download from monitter.com.


USAGE:

set your picasa's username and public albumid as follows.

<div class="picasatter" id="album1" user="username" album="albumid"></div>
